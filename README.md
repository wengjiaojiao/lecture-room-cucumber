# lecture-room-cucumber

运行步骤
--------

1.克隆项目到本地

```
git clone git@github.com:wengjiaojiao/lecture-room-cucumber.git
```

2.进入lecture-room-cucumber目录，执行

```
npm install
```

下载依赖包

3.安装mysql数据库，

```
brew install mysql
```

4.创建名为bookmarks的数据库，用户名是root，密码为空，连接数据库。

5.输入命令，

```
node app.js

```

启动服务器,打开浏览器，在地址栏中输入 localhost:3000 即可查看网页
